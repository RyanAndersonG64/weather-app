# Vanilla JS Bundler
A repo template built with Vite.

## Initial Setup for GitHub Pages deployment
* in vite.config.js change 'base' value to the name of your repository
* in GitHub: repo: settings: pages: branch: change from /root to /docs

## To Run Locally
* $ nvm use
* $ npm install
* $ npm run dev
